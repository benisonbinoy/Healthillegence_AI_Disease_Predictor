# Model Storage Directory
# This directory contains trained machine learning models for the Healthiligence application

Models stored:
- diabetes_model.pkl - Random Forest model for diabetes prediction
- kidney_model.pkl - Random Forest model for kidney disease prediction
- liver_model.pkl - Random Forest model for liver disease prediction
- malaria_model.h5 - CNN model for malaria detection from cell images
- pneumonia_model.h5 - CNN model for pneumonia detection from chest X-rays
- model_info.json - Contains accuracy and training information for all models
